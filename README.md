# whisper2.0

whisper2.0使用neteasecloudmusicapi提供的网易云音乐api，在1.0基础上进行改版

截至目前功能还不完善，后续有时间再慢慢补上

[在线演示](http://118.193.37.162/index.html#/index)



手机扫码访问 ![](./1625558804.png)

- 音乐搜索
- 音乐播放
- mv播放
- 歌单展示
- ...
